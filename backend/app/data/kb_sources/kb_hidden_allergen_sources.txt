Тема: Скрытые источники аллергенов

Назначение для MVP:
Файл нужен для обработки ингредиентов, в которых аллерген может быть неочевиден по простому русскому названию.

Скрытые или менее очевидные формы:
- whey, casein, buttermilk, curds -> молочные белки
- albumin, meringue, egg wash -> яйца
- fish sauce, Worcestershire sauce (иногда с анчоусами) -> рыба
- tahini -> кунжут
- soy lecithin, textured vegetable protein, miso, tofu -> соя
- seitan -> пшеница / глютен
- marzipan -> миндаль
- pesto -> часто содержит древесные орехи и/или сыр
- surimi -> может содержать рыбу и добавки с аллергенами
- natural flavors, sauces, breading mixes -> требуют осторожности при неполном составе

Что учитывать:
- «Contains» и «may contain» не одно и то же
- если на этикетке есть Contains, это означает наличие соответствующего основного аллергена среди ингредиентов
- advisory statements типа may contain интерпретируются отдельно и не должны смешиваться с прямым составом
- для MVP можно поддерживать два уровня риска: explicit ingredient и possible hidden source

Рекомендации:
- хранить словарь скрытых аллергенных синонимов
- для полуфабрикатов, соусов и готовых смесей помечать риск как «требует проверки состава»
- не делать ложное заверение безопасности, если состав неполный
