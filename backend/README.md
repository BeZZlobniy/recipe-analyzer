# Backend

Backend-часть проекта содержит:

- `app/` — основную серверную реализацию FastAPI;
- `demo/` — демонстрационные рецепты и сохраненные JSON-результаты;
- `scripts/` — служебные утилиты для прогонов, оценки и отчетности.

Локальные отчеты, временные базы данных и generated-артефакты не версионируются.

Актуальная общая документация и структура проекта описаны в корневом [README](../README.md).
